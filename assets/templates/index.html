How to embed widget on your website

1. Add <script type="text/javascript" src="http://www.tursh.net/widget/"></script> to the head or below the body of your website.

2. Add <div id="tursh_widget" widgetType="short" customStyle="false"></div>

* id="tursh_widget" is compulsory. The div with that id will contain the widget's html. You have to make sure you don't have more than 1 element with that id.

* widgetType="short" or widgetType="long". Default is short. If no widgetType is specified or an invalid value was specified, the default value is used.

* customStyle="false" or customStyle="true". Default is false. It is used to specify whether you want to apply a customStyle or not. If no value is specified or an invalid value is specified, the default value is used.
